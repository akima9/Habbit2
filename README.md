# 금연인 (안드로이드 부분)

**하루에 하나씩 줄여가는 금연 보조 안드로이드 애플리케이션입니다.**

**[ 개발환경 ]**

**· 개발인원 :** 1명

**· IDE :** Android Studio

**· 인프라 :** AWS EC2(Ubuntu)

**· 개발언어 :** JAVA

**· 웹서버 :** Apache

**· 데이터베이스 :** MySql

**· 상세내용**

1. 전체적인 UX/UI : BottomNavigationView & Fragment를 사용하여 메뉴와 3개의 화면(홈, 통계, 계정)을 구현하였습니다. CalendarView를 사용하여 달력 UI 구현하였습니다.

2. 자동로그인 : sharedpreferences를 사용하여 자동로그인 구현하였습니다.
